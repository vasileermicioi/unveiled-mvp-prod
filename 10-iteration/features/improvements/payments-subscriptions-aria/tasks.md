# Umbrella Tasks — Payments & subscriptions ARIA pass

The umbrella's `tasks.md` lives at
`openspec/changes/payments-subscriptions-aria/tasks.md` and is the
source of truth for the umbrella-level task list. Each per-row
`<row-slug>/tasks.md` mirrors the row's checklist from the
umbrella's `tasks.md` so the per-row Definition of Done can be
checked off independently.
